Description

StDrawingArea is similar to ClutterCairoTexture in that it allows drawing via Cairo; the primary difference is that it is dynamically sized. To use, connect to the "repaint" signal, and inside the signal handler, call st_drawing_area_get_context() to get the Cairo context to draw to. The "repaint" signal will be emitted by default when the area is resized or the CSS style changes; you can use the st_drawing_area_queue_repaint() as well.
